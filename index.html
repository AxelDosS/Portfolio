Création site cinéma

Le projet consistait à concevoir et développer un site web pour un cinéma, offrant une expérience interactive aux utilisateurs et permettant à l'administrateur de gérer les films, les réalisateurs, les salles, les horaires et les genres. Voici les détails du projet :
Technologies Utilisées : HTML, CSS, PHP, MySQL (base de données SQL), phpMyAdmin
Sections Principales :
Accueil : La page d'accueil présente une introduction au cinéma, peut-être avec des informations sur les prochains films à l'affiche, des offres spéciales, etc.
Affichage : Cette section permet aux utilisateurs de parcourir les films disponibles, les réalisateurs, les salles, les horaires et les genres. Chaque film est présenté avec des détails tels que le titre, l'affiche, le réalisateur, la durée, le genre, etc. Les utilisateurs peuvent consulter les horaires de projection et les détails des films.
Insertion : Dans cette section, les utilisateurs doivent se connecter pour pouvoir ajouter, retirer ou modifier des films, des réalisateurs et des projections. Un système de gestion des utilisateurs et des sessions est nécessaire pour authentifier les utilisateurs et leur donner accès à ces fonctionnalités d'administration.
Fonctionnalités Clés :
Affichage des films : Les utilisateurs peuvent parcourir les films disponibles et voir les détails de chaque film.
Affichage des réalisateurs : Les utilisateurs peuvent consulter les informations sur les réalisateurs des films.
Affichage des salles et horaires : Les utilisateurs peuvent voir les horaires de projection des films dans différentes salles.
Affichage des genres : Les utilisateurs peuvent consulter les genres des films disponibles.
Insertion, mise à jour et suppression : Les utilisateurs autorisés peuvent ajouter de nouveaux films, réalisateurs et projections, mettre à jour les informations existantes et supprimer des entrées.
Résultats :
Ce projet a permis de mettre en pratique les connaissances en développement web et en gestion de bases de données. Il a également fourni une expérience pratique dans la création d'un système d'administration sécurisé pour gérer les données du cinéma.

Magasin de musique en ligne

Le but de ce projet est de consevoir et développer un site e-commerce de musique en ligne, impliquant l'utilisation de HTML, CSS, JavaScript, PHP et MySQL.
Il fallait que le site offre une expérience utilisateur intuitive avec des fonctionnalités de connexion pour les clients et les employés, gestion des comptes clients, navigation dans le catalogue musical, et gestion des ventes et données par les employés.
Ce projet a pour but de démontrer une maîtrise des langages web, du modèle MVC, et de l'interaction avec une base de données SQL.
Ce projet était surtout le premier que j'ai réalisé en équipe, et il m'a permis d'apprendre davantage sur le développement de sites internet.
En plus de cela, il nous a également permis de créer un site en répartissant efficacement le travail entre les membres de l'équipe.
